# Update the cached copy of the system requirements database

Update the cached copy of the system requirements database

## Usage

``` r
sysreqs_db_update()
```

## Details

If the the cached copy is recent, then no update is attempted. See the
`metadata_update_after` `man_config_link("configuration option")`.
